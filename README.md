# Responsive Single Page Application was bootstrapped with Create React App. - Recipe App

## Frameworks and Libraries:

- React 16.8.4
- react-router-dom
- axios

## Providers:

- Using firebase as rest api
- Image preview and upload with firebase
- Nested forms
- CRUD
- Daynamic rating stars
